# 数据字段说明



### 文件命名规则

| 后缀 | 含义 | 示例 |
|------|------|------|
| `_ohlcv` | 日 K 线（Open / High / Low / Close / Volume）等 | `000001_SZ_20250701_20260601_ohlcv.csv` |
| `_ind` | 每日指标（Indicators） | `000001_SZ_20250701_20260601_ind.csv` |

统一格式：`{股票代码}_{起始日期}_{结束日期}_{后缀}.csv`

---

## 1. 日 K 线（`_ohlcv`）

文件命名示例：`000001_SZ_20250701_20260601_ohlcv.csv`

| 英文字段 | 中文名称 | 说明 | 单位 |
|----------|----------|------|------|
| `ts_code` | 股票代码 | Tushare 格式，如 `000001.SZ` | — |
| `trade_date` | 交易日期 | 格式 `YYYYMMDD` | — |
| `open` | 开盘价 | 当日开盘成交价 | 元 |
| `high` | 最高价 | 当日最高成交价 | 元 |
| `low` | 最低价 | 当日最低成交价 | 元 |
| `close` | 收盘价 | 当日收盘成交价 | 元 |
| `pre_close` | 昨收价 | 前一交易日收盘价 | 元 |
| `change` | 涨跌额 | `close - pre_close` | 元 |
| `pct_chg` | 涨跌幅 | 相对昨收的百分比变化 | % |
| `vol` | 成交量 | 当日成交股数 | 手（1 手 = 100 股） |
| `amount` | 成交额 | 当日成交金额 | 千元 |

---

## 2. 每日指标（`_ind`）

文件命名示例：`000001_SZ_20250701_20260601_ind.csv`

| 英文字段 | 中文名称 | 说明 | 单位 |
|----------|----------|------|------|
| `ts_code` | 股票代码 | Tushare 格式，如 `000001.SZ` | — |
| `trade_date` | 交易日期 | 格式 `YYYYMMDD` | — |
| `close` | 收盘价 | 当日收盘成交价 | 元 |
| `turnover_rate` | 换手率 | 成交量 / 流通股本 | % |
| `turnover_rate_f` | 换手率（自由流通） | 成交量 / 自由流通股本 | % |
| `volume_ratio` | 量比 | 当日成交量 / 近期平均成交量 | 倍 |
| `pe` | 市盈率（静） | 总市值 / 最近年报净利润；亏损时为空 | 倍 |
| `pe_ttm` | 市盈率（TTM） | 总市值 / 近 12 个月净利润 | 倍 |
| `pb` | 市净率 | 总市值 / 净资产 | 倍 |
| `ps` | 市销率 | 总市值 / 最近年报营业收入 | 倍 |
| `ps_ttm` | 市销率（TTM） | 总市值 / 近 12 个月营业收入 | 倍 |
| `dv_ratio` | 股息率 | 按去年分红计算的股息率 | % |
| `dv_ttm` | 股息率（TTM） | 按近 12 个月分红计算的股息率 | % |
| `total_share` | 总股本 | 公司发行的全部股份 | 万股 |
| `float_share` | 流通股本 | 可在二级市场交易的股份 | 万股 |
| `free_share` | 自由流通股本 | 剔除大股东限售等后的可交易股份 | 万股 |
| `total_mv` | 总市值 | 收盘价 × 总股本 | 万元 |
| `circ_mv` | 流通市值 | 收盘价 × 流通股本 | 万元 |
